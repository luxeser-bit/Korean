# HanTalk Korean

React MVP приложения для изучения корейского с русской и английской локализацией.

Интерфейс сделан в формате минимального каталога курсов: верхняя навигация, заголовок курса, список юнитов, панель выбранного юнита и футер. Проект не копирует бренд, ассеты или учебные тексты Teuida.

## Запуск

```bash
npm install
npm run dev
```

## Деплой на GitHub Pages

Проект уже содержит workflow `.github/workflows/pages.yml`. После пуша в `main` или `master` GitHub Actions соберет React/Vite проект и опубликует папку `dist`.

В репозитории GitHub нужно включить Pages:

1. Откройте `Settings -> Pages`.
2. В `Build and deployment` выберите `Source: GitHub Actions`.
3. Запушьте проект в `main` или `master`.

## Что внутри

- `src/App.jsx` - React UI, каталог юнитов, выбор языка и панель выбранного юнита.
- `src/data/lessons.js` - юниты курса и переводы интерфейса.
- `src/styles.css` - адаптивный интерфейс для desktop, tablet и mobile.
- `Start HanTalk Korean.cmd` - launcher для запуска с рабочего стола.
